# موقع رواية الوريث الخامس

موقع قراءة عربي تفاعلي يعمل بالكامل كصفحات ثابتة، ومناسب للرفع على GitHub Pages أو Vercel.

## التشغيل المحلي
- افتح `index.html` مباشرة للقراءة.
- لتفعيل وضع التثبيت والعمل دون إنترنت، شغّل المجلد على خادم محلي، مثل:
  `python -m http.server 8000`

## المزايا
- جميع فصول الرواية كاملة.
- حفظ تلقائي لموضع آخر فصل.
- وضع ليلي وورقي، وتغيير حجم الخط.
- قائمة فصول وبحث.
- مؤثرات بيئية مولدة داخل المتصفح، من دون موسيقى أو ملفات صوت خارجية.
- تصميم متجاوب للموبايل والكمبيوتر.
- دعم PWA والعمل دون إنترنت بعد الزيارة الأولى عند الاستضافة.
